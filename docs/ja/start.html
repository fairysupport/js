fairy support jsのインストール

ここからzipファイルをダウンロードし、適当なフォルダに置き、解凍してください
解凍したディレクトリ内のpackage.jsonがあるディレクトリにコマンドラインで移動 

npm install

と入力(npmは事前に使えるようにしておいてください)
完了

起動

コマンドラインで

npm run watch_local

watchなのでコマンドは実行したままとなります

また別のウインドウからコマンドラインで

npm run server

ローカルのサーバーが起動し、ブラウザにsampleボタンだけがある画面が表示されます
sampleボタンをクリックするとHello fairysupportとalertが出力されれば成功です
停止する場合はnpm run watch_localしたウインドウ、npm run serverしたウインドウ両方でCtrl+Cを入力して停止してください
